# Multi-Agent Cleaning System

## Overview
A grid-world multi-agent simulation project for comparing different decision-making strategies in a collaborative cleaning task.

## Objective
The goal is to analyze how Random, A*, Q-learning, and shared Q-learning agents perform in terms of path efficiency, task completion, and collaboration.

## My Contribution
- Built the 10x10 simulation environment
- Implemented RandomAgent, AStarAgent, QLearningAgent, and SharedQLearningAgent
- Designed shared-map and shared Q-table collaboration settings
- Compared strategies using task completion steps, efficiency, and convergence patterns

## Tech Stack
Python, Reinforcement Learning, A*, Q-learning
